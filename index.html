FAQs

How do I use a manual location?

Follow these steps. 1. Go to the settings page. 2. Turn the manual location switch to on (should be mostly red). 3. Enter a manual location in the text box. (format: City, State)

How do I find a new random restaurant?

Click the Find button on the top toolbar.

Can I set a distance?

Yes, in the settings page change the slider to a desired maximum distance.

Where does the restaurant data come from?

Where to Eat uses Yelp API to find and display info about restaurants near the user.

No restaurants are found around me. What do I do?

Recommended steps to take are: 1. Check that you have location services enabled. 2. If using manual location, verify a location is entered. 3. Click the find button again. 4. Verify you can connect to the internet. 5. Lastly, report a problem here.
